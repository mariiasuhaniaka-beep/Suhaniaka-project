# Music Catalog App

## Description
Full-stack application for managing music tracks.

## Features
- View tracks
- Add new track
- Delete track
- REST API integration

## Technologies
Frontend:
- React (Vite, TypeScript)

Backend:
- NestJS

## Installation

### Backend
cd backend
npm install
npm run start

### Frontend
cd frontend
npm install
npm run dev

## API Endpoints
GET /tracks  
POST /tracks  
DELETE /tracks/:id
